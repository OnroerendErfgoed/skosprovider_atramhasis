.. _glossary:

Glossary
========

.. glossary::
   :sorted:

   Atramhasis
    `an online SKOS editor <http://atramhasis.readthedocs.org/en/latest>`__.
    It allows a user to create and edit an online thesaurus or vocabulary adhering to the SKOS specification through a simple web interface.
    This allows any user with access to a web browser to consult the thesauri and if so wanted, to edit them.

   SKOS
    `Simple Knowledge Organization System <http://www.w3.org/2004/02/skos>`__. An
    general specification for Knowledge Organisation Systems (thesauri, word 
    lists, authority files, ...) that is commonly serialised as :term:`RDF`.

   URI
    A `Uniform Resource Identifier`.

   URN
    A URN is a specific form of a :term:`URI`.
